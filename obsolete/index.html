    下午

    明夷虫

    07-16

    我在想这个下午该怎么度过。因为这是一个非常陌生的下午。虽然很久以前我很熟悉它，但是当它重新回来的时候，像是我阔别多年的朋友或恋人。由于我从未有过这样的朋友，更惶说恋人，所以我不知道该怎样与它相处。

    嗨，你好啊，下午！

    我的确试着很小心地跟它打了个招呼。它没有回应。这是肯定的。近五个月以来，我经历过的这些下午，似乎也永远不再回应我了。整个上午一直想哭。只是昨晚悄无声息地哭过了，现在不太好意思。

    这个世上也许不会有人再比我可怜的了。饿死的人也许都比我幸福。至少饿得失去力气的人，没有心情去伤心，去心痛。他们总是抱着下一秒会有人施舍给他一个馒头的希望。至少他们还有希望。即使他们死去，也是死得其所，而不是像我这样，活非其所。我熟悉的昨天没有了。不熟悉的现在摆在我面前，我不知该如何度过。那个没有什么希望的明天还在等着我。我像是被困在无限循环的车站里的那个黑客，无论他怎么走，也走不出车站，而且也没有机会上车。

    杀人犯，即使不被世人唾弃，也要判死刑。伤心犯，法律不禁止。当我刺伤了一个人的心之后，没人唾弃我。我的几个网上的朋友发现了我的异常，反而安慰我。这也许是最大的惩罚。好了，我接受。但是，我该如何度过这个下午。到现在为止，我这个正在用来写字的某知名 APP，我已经反复卸载又安装了五次。我的一片空白的微信已经反复看了上百次。那个被我刺伤的人似乎决定不再理睬我了。

    我终于理解了，冷漠甚于死刑；也理解了，哀莫大于心死。我终于开始赞成废除死刑，设立冷漠刑，也终于开始理解那些为底层人民鼓与呼的仁人志士了……没错，脏的，可以变成干净的。我的心也会变。我害怕的蟑螂，现在如果有人将它放在我身上，我会无动于衷地坐在那里。可是，我变心了，也没用。我依然不知道该如何度过这个下午。我甚至觉得圣经里的结论也开始正确，「回答柔和，使怒消退；言语暴戾，触动怒气」。我本以为，我这么爱她，肯定不会伤害她，但是当我控制不住我的语言时，依然刺伤了她。

    我应该继续翻译概率论。在以往属于上午的事，用来打发一些时间，等待那个被我刺伤的人出现。她不会再出现了。不对，她会出现，只是不会再为我出现。我应该看一会冰与火之歌，然而打开书，很久也没有看完一页。我还是控制不住去看手机。虽然我知道，她不会出现了，但还是控制不住啊。于是，我又刷了几十次手机。我像个快要饿死的人在等待一小块面包。

    微信有新消息提示的声音，但我故意关掉了。我不想听到声音，打开微信，结果发现是别人发来的信息。这是上百个昨天养成的习惯，在这样的一个下午，我改不了。现在还刚刚 15 点 19 分。她还没有起床吧。我知道昨天晚上，我给她发的信息，她应该都看到了。她没再回复我。也许是失望，也许是累了，也许是想干脆就把我忘了。她怎么想的，我并不清楚。昨天一整天，她都懒得和我说话了。所以，我总觉得她生气了，但是也没法去肯定。也许她并没有打算要冷漠对我吧……只是因为太累，很早就睡了。

    你都开始这样骗自己了？

    不对，我没有骗自己。事实上，她怎么想的，我的确不知道。一切都是我一厢情愿地那样认为。我好像知道了什么。她说聊天很累，必然指的是和我聊天很累。

    其实我一点都不会聊天。我的思路经常会莫名其妙地跳来跳去。和每个人聊天，即使很熟悉的人，我也没法改变这个毛病。我很少去关心别人怎么想，只是任由自己不断变着的想法去说话。一直以来，也许是因为我很爱她，所以很尊重她，大多数时间，我还是会很认真地听她说话。也正是因为自己不太会聊天，所以要真得整理思路，我都是要靠写文章。文章写得慢，也很容易修改，但说话不同。说错的话，就很难收回来。说得越多，错的也就越多。如果她能原谅我，我可以跟她学着去聊天啊……

    又在做梦。她不会原谅我了。唉，我也不知道，她为什么会生气。仅仅是因为我言语上冒犯了她喜欢的人么？我觉得很有可能是她逐渐发现了我是个无趣的人。我不知道该怎样让自己有趣，毕竟我不是个爱看书的人，何况我看的书也都挺无趣的。更何况，她喜欢的那个人，一说话就是满腹经纶，我一说话就得去查书。

    我还是想想该怎么度过这个下午吧。到现在，也还刚刚15 点 33 分。我去翻译概率论吧。我再看一下手机……脑子真是有病了。微信网页打开了，有消息就该提示了。我还是把网页关上为好。否则，收到别人的消息，我又会失望。没有希望，总比失望更好一些吧。

    我真的是这个世上最可怜的人了，也许只有这么一个下午。事实上，倘若今天下午我不知道该怎么过，那么明天下午也是一样，也许这就是她那天晚上说的西西弗斯什么的……我翻译概率论去了。不过，这样乱七八糟地写东西，看上去也不错，现在已经被我蹭到了 15 点 47 分了。可是，让我的伤心的事就这么少，能写几天呢？有点困了，我先睡一会。睡觉是个消耗时间的好办法。

    睡不着。只要闭上眼，伤心就会像黑色的潮水般涌来，忍不住又想哭。可是我昨天哭过了。15 点 51 了。她快要醒了，也许已经醒了。这也是她的这几个月以来的第一个下午吧，但愿她不会像我这样。应该不会。她有一次说我偏激，爱走极端。我觉得我现在就很极端的了。我还没自杀啊，也不算极端。不过，我也不会自杀。自杀也是对那些关心我的人的漠视。我杀了自己，也可能会让他们像我现在这样痛苦。

    如果她觉得这样的下午与平常也没什么不同。我是该为她高兴，还是该为自己伤心？都会有。但是，她若能做到这样，那我替她高兴，也就是没必要的了。我该为自己伤心。不对，倘若她如此，那就意味着我并没有刺伤她的心。那么，是谁的心被刺伤了？我特别想问一下她，有没有伤心。我试着问一下吧，反正她也不打算理我了，我就问一句。

    我问她去了……在等答案。我先翻译概率论……

    她不想回答我的任何一个问题了…… 16 点 21 分……尽管一天过去了，她依然是和我说一句话都觉得很累，或者对我极其厌恶。我很想被她拉入黑名单，让我死掉和她说句话的幻想，总比给我幻想却永远不让我实现更为仁慈。我依然抱有一丝的幻想，很诚恳地请求她给我一个答案，无论什么时间都可以。

    算了，我不要答案。先假设她没伤心吧。她没伤心，但我如此伤心，而且我又觉得我伤了她的心。这个关系很奇怪。本来是我觉得自己伤了她的心，从而导致她开始漠视我。我的伤心则是因为她的漠视引起的。现在前提不成立，所以，最终结论也不成立，即我的伤心不是她的漠视引起的。

    从另一个角度来看，假设她也伤心了，而且的确是我伤了她的心。那么，我为什么要伤她的心呢？我爱她，无论如何我都不会去伤害她，即使她对我如此漠视，我依然没有这样的想法。那么，她伤心，只能是误伤。鉴于她到现在也没有告诉我是否伤心，所以，即使她是误伤，我依然不知情。也就是说，我的伤心也不是因为我伤害了她而引起的。

    唉……我的伤心，不是她的漠视引起的，也不是因为伤害了她而引起的，那么，我为何伤心？

    答案只有一个了。我在伤害我自己。整件事情都是我一个人一点一点策划出来，而她不过是卷入其中的无辜者。

    她说的是对的，若不是放大自我，就什么事都不会有。另外，从昨天到刚才我得到答案之前，我一直觉得她的漠视仅仅是出于不再爱我。事实上，这与她爱不爱我也没有关系。即使她爱我，她也是有权力终止此爱，因为她爱的并非那个放大了自我的我。

    16 点 49 分。我弄明白了一个问题。但是随之而来的是，我为什么如此处心积虑伤害我自己？我出了问题。老板来了，我只好停止思考。不过，也没有关系，我还有很多时间。

    概率论又译完了一节。17 点 36 分，老板还没走，但是貌似也不会找我议事。我可以继续去思考刚才的问题，我出了什么问题……这件事必定与爱情有关。刚开始去想，又被老板打断了。

    18 点 00 分，我处心积虑伤害我自己，这似乎没有道理，其实是有道理。当我想到这个道理之后，有些毛骨悚然。我之前和她说过，我最近总是有一种自己被捆在祭台上的感觉。我觉得周围一切的人，都是各种广义宗教的教徒，我们不过是他们的祭品。在我处心积虑策划的这件事情里，我是祭品，但我也是祭司。就像那些信奉神灵的宗教一样，祭司向神灵献祭活人的整个过程，无需理睬祭品的感受，只需要践行自己作为神职人员的义务。这在形式上来看，便相当于祭司在处心积虑的将祭品捆绑在祭台上。我就是这样，被那个信奉爱情的祭司，送上了祭台。

    我如此伤心，我又如此阴暗。当我想明白这些，我依然很想大哭一场。事实上，去年我就发现了爱情里的献祭。但是，当这个爱情落到我的头上的时候，我却把这些忘得一干二净。当时我写了这些东西：

    无论是单恋，还是两情相悦，它们的共同之处在于，所爱之人在自己心目中应当足够美好。这个时候，我没那么多闲工夫去考虑献祭，耍流氓，传播基因，分泌多巴胺，结婚……我爱的是这种美好。只有这种美好是客观存在的，其它任何东西在这种美好存在的时刻，都是不存在的，它们不过是在这美好飘逝之后而强加追忆，本质上都是献祭，只是祭品有所不同。

    不少人，在爱上一个人的时候，认为这个人是自己的全部，一旦失去，便痛不欲生，从而干出一些难以预料之事。这种行为可视为心灵投射，认为自己所臆想的东西是客观存在的，不存在也要创造出来让它存在。

    爱情是一种功能，不是一种形态。它可以让我看到一个美好的自己，或者自己美好的一面。认为自己所爱的是一个具象的人，进而要求这个人的所作所为必须对得起自己的爱，这是最大的移情别恋——将对自己的爱，转移到了对方身上。

    我之前的观点是对的，我亲身验证了它的正确。我感受到的伤心与痛苦，是被我自己献祭了自己的结果。她应该没有，因为她没有我这么变态……

    我现在很愧疚。是的，现在的我一点都不值得任何人去爱。也许我的心应该去死掉才对。我不知道自己还能不能把心再变回去，我会努力的。我一直都很爱她，但是，我应该从她那里看到自己的美好，而不是内心阴暗到近乎失去理性的人。

    我想明白了，但是我也失去了。她走了。我无处可去。现在，祭司死了，但我依然被捆在那里。我最想求助的人是她啊。

    爱情

    明夷虫

    07-16

    我一贯自以为是，思考过许多与人生观有关的问题，唯独未曾认真思考过爱情。估计是我的爱情太多了，一时半会也思考不出来什么。

    文学作品写爱情，从诗经就开始写，到现在已经两千多年了。人类想明白了么？

    肯定还没有，否则就不会有那么多人还在写爱情小说。不仅还在写，而且内容也逐渐细化了起来，校园爱情，乡村爱情，都市情感，同性爱情，情色抑或色情，甚至还有人兽爱情——

    我看过次数最多的三篇爱情小说，《一个陌生女人的来信》，《茵梦湖》以及《伊豆的舞女》，从这些小说里，只学会了情绪上的忧伤，以为爱情就是一种令人忧伤的东西。这种忧伤来自于，眼前这么多美好的姑娘，得不到她们的芳心，也就罢了……问题是，连身体也得不到啊。

    有时看到那么多的男人在电视剧里抱怨或愤怒，「我得到的只是你的身体，而不是你的心」，就觉得他们很蠢，他们不珍惜自己的女人的身体。当然，也有些女人在电视剧里很勇敢地说，「你只能得到我的身体，得不到我的心」，这时又觉得这些女人也很蠢，因为她们也不珍惜自己的身体。

    不过，看起来，爱情这种东西，与外在无关。

    真的是这样吗？

    不知道。

    只知道，还是要看脸的。不仅要看脸，还要看身材。不仅要看身材，还要看家世。不仅要看家世，还要看工作，看爱好，看星座……要看的东西这么多，还有多少时间去看心呢？再说了，心，是啥，又该如何看？

    还有一些人会有一些奇怪的理论。譬如，女人爱一个男人，就会为他生个孩子，而且会很爱这个孩子。于是，为了爱情，还要再搭上孩子。

    好久没用我的「如无必要，勿设变化」剃刀了。既然要看的东西这么多，是这些东西不存在，还是爱情本身就不存在？

    爱情，需要这么多的祭品的爱情，存在么？

    哲学家很少讨论爱情。他们到现在连人为啥要活着还没搞清楚，没太多时间思考爱情。

    那么科学家呢？他们往往有意无意地暗示，爱情不过是动物传播基因的本能而已，不过是脑子里的多巴胺分泌过多而已。

    社会学家呢？爱情，不过是维护社会安定耍流氓而已，而且美好的爱情也会死，死于婚姻这座坟墓。

    太过于神圣的东西，其命运终归躲不过受践踏，变得一文不值。

    那么，就干脆让这种东西彻底消失吧。让它不存在。这样就可以不献祭，也可以不践踏。大家可以把时间节省下来，去干一些别的无聊的事。这种事，大家都不怎么想做，因为实在太无聊了。所以，我来。

    「如无必要，勿设变化」，这柄剃刀，它的背后其实有着更深刻的原理，那就是，复杂的东西，它的原理必定是简单的，否则它不能维持自身的长久存在。

    爱情如此复杂，又如此长久存在，那么它的原理必定是简单的。

    对爱情最简单的描述，应该是，「窈窕淑女，君子好逑。求之不得，辗转反侧」，这是单恋。或者，「有女怀春，吉士诱之，舒而脱脱，无感我帨，无使尨吠」，这是两情相悦。要研究爱情，须从这种原始描述开始，而不是从现代被一堆乱七八糟的东西囚禁起来的描述开始。

    无论是单恋，还是两情相悦，它们的共同之处在于，所爱之人在自己心目中应当足够美好。这个时候，我没那么多闲工夫去考虑献祭，耍流氓，传播基因，分泌多巴胺，结婚……我爱的是这种美好。只有这种美好是客观存在的，其它任何东西在这种美好存在的时刻，都是不存在的，它们不过是在这美好飘逝之后而强加追忆，本质上都是献祭，只是祭品有所不同。

    这种美好，后来去了哪里？

    哪里都没去，它还在我心里。我甚至没法用文字或其他形式将它抛弃。所以，我很难向任何一个人描述它，更不要说去写爱情小说或情诗了。以至，但凡遇到自己极欣赏的女性，我只能简单地冠之以女神的花环。倘若我心里缺乏这种美好，可能会无法欣赏一切女性。幸好，我的女神很多，她们甚为可爱。只是这种可爱，她们自己无法想象。

    我觉得，爱情，一旦有了，它就会变成生命的一部分。婚姻埋葬的不是爱情，而是我们自己。我们自己也可能不是婚姻埋葬了，而是我们自己闭上了眼睛，然后抱怨天太黑。可能问题始终出在，我们并不像自己所以为的那样了解自己，但是又坚信世上只有自己最了解自己。

    人是无法了解自己的。倘若你真的很相信自己最了解自己，那么从小到大，这个所谓的最了解的自己，不知变了多少次了。倘若你真的相信自己，那你就是个一成不变的人，这是一种病，幼稚病。

    不少人，在爱上一个人的时候，认为这个人是自己的全部，一旦失去，便痛不欲生，从而干出一些难以预料之事。这种行为可视为心灵投射，认为自己所臆想的东西是客观存在的，不存在也要创造出来让它存在。世俗的爱情，需要按照这种套路去演绎，否则写小说的，写剧本的，拍电视/电影的，都不太好混。

    爱情是一种功能，不是一种形态。它可以让我看到一个美好的自己，或者自己美好的一面。认为自己所爱的是一个具象的人，进而要求这个人的所作所为必须对得起自己的爱，这是最大的移情别恋——将对自己的爱，转移到了他人身上。

    爱情这种功能，可以让我美好。无论我爱的人是不是还在我的身边，我都可以保持这种美好。

    那么，两情相悦呢？

    彼此从对方身上看到了自己的美好，而且愿意做对方的镜子。

    爱情，也许就是这么简单。

    昵称

    给心爱的人
    取了一个昵称
    没有告诉她
    不会告诉任何人
    我隐藏了一扇门
    门上纹饰她的气息
    悬挂一把失去了的小刀
    再用一把锁
    锁上最后的依恋
    我在门后
    安放了美丽世界
    最后一把钥匙
    我丢在了她的昵称里
    昵称丢在心里
    心无处可藏
    风里破碎
    雨里哭泣
    阳光里微笑

    Nikola 是我用了 3 年的静态网页生成工具，这个 Github 主页上的所有页面皆由它生成。它的安装与基本用法可参考《Nikola 入门文档》，对于 Linux 环境，建议在 virtualenv + python 3 环境中安装它。它的更详细的用法，见《Nikola 手册》。本文仅记录我在使用 Nikola 过程中遇到的一些小问题以及不甚高明的对策，仅作备忘之用。

    自定义首页

    Nikola 会自动生成首页，将 10 近期的文章内容收集到一个页面，该页面对应的文件为 index.html。这份文件默认会被放到站点的根目录里，所以，它就成了站点的首页。我觉得 Nikola 这个功能很鸡肋，因为我想自己定义首页，而不是这种自动生成的东西。该怎么办呢？

    首先，在 conf.py 文件里，将 INDEX_PATH 设为

    INDEX_PATH = "blog"

    这样，Nikola 就会将这个自动生成的 index.html 文件放到 站点根目录/blog 目录里，从而令它失去了担当站点首页的资格。理论上，也可以随便取个目录名来用，例如 ·

    INDEX_PATH = "obsolete"

    用这种乾坤大挪移的办法，算是将这个鸡肋的站点首页「消灭」了。接下来，就是建立一个自定义的首页，以后手工来维护它。方法很简单，只需要在 PAGES 目录，任意建立一份源文件，将其 slug 设为 index.html 即可。

    PAGES 目录，就是在 conf.py 文件中设定的目录。例如，我的设定是：

    PAGES = (
    ("stories/*.md", "", "story.tmpl"),
)

    这就意味着，我可以在 stories 目录建立 slug 设为 index.html 的源文件（至于它对应的文件名是什么，并不重要）。假设这份源文件是 Markdown 格式，那么所谓 slug 就是文件头中的 slug 条目，例如：

    <!-- 
.. title: 
.. slug: index
-->

    剩下的内容可自由定义。

    当网站内容重新发布时，这份源文件就会被转化为网站根目录中的 index.html 文件。因为 Nikola 就是这样对待 PAGES 目录中的源文件的，会把它们「平铺」在网站的根目录里。
